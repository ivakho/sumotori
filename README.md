# Cars Selector

## Сайт приложения для тестирования

https://ivakho.github.io/sumotori/

## Описание

Небольшое SPA-приложение для подбора «машины мечты» в несколько шагов.

Пользователь:
1. Выбирает **марки**, **модели** и **кузова** (с зависимостями).
2. Подтверждает выбор.
3. Получает **рандомный результат** в формате  
   **`МАРКА МОДЕЛЬ КУЗОВ`**  
   (совместимость не учитывается).

Приложение реализовано как пошаговый процесс (Stepper), без перезагрузки страницы.

## Стек технологий

- **React + TypeScript**
- **Mantine UI** — компоненты и Stepper
- **TanStack Query** — загрузка и кеширование данных
- **SCSS Modules** — стили
- **Vite** — сборка
